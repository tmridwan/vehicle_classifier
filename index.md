# Vehicle Classification Project

## Environment Setup

### Requirements
- Python 3.8+
- fastai library
- PyTorch
- Jupyter Notebook

### Installation

```bash
pip install fastai
pip install torch torchvision
pip install jupyter
```

### Dataset
This project uses vehicle images to classify different types of vehicles.

## Usage

1. Clone the repository
2. Install dependencies
3. Run the Jupyter notebook for training
4. Use the trained model for predictions

## Model Details
- Framework: fastai
- Architecture: ResNet-based transfer learning
- Task: Multi-class vehicle classification
